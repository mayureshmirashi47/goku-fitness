# ⚡ Goku Fitness Centre

An ultra-interactive, premium fitness web experience designed to push training past mortal limits. This project blends high-performance frontend design with cinematic 3D elements, featuring a fully interactive 3D model of Goku and immersive custom animations.

---

## 🔥 Features

*   **3D Hero Element:** Features a highly detailed, floating 3D model (`goku.glb`) that seamlessly pans and rotates as you scroll.
*   **2D Energy Aura:** An interactive background particle canvas that responds dynamically to mouse movement, mimicking a surging Saiyan aura.
*   **Liquid Glass UI:** Built with sleek neon-accented glassmorphism layouts, liquid-sheen custom buttons, and a tailored dual-ring cursor tracking system.
*   **Power Level Counter:** An animated live counter that ramps up to break past 9,000 as an introductory text timeline triggers.

---

## 🛠️ Tech Stack

*   **Core Structure:** HTML5 / CSS3
*   **3D Graphics:** Three.js & GLTFLoader (Object rendering & lighting)
*   **Component Architecture:** React (JSX structure for modular environments)
*   **Motion & Scroll Handling:** GSAP (GreenSock Animation Platform) & ScrollTrigger
